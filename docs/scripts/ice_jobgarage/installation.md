# Installation

## Requirements
- FiveM server
- ESX (or compatible job system)
- ox_lib (optional / if your version uses it)

## Steps
1. Place `Ice_jobgarage` in `resources/`
2. Add to `server.cfg`:
   ```
   ensure Ice_jobgarage
   ```
3. Configure `config.lua`
4. Restart the server
